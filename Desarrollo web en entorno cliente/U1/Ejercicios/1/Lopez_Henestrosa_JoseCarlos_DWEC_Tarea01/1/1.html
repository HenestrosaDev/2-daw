b) ¿Por qué es conveniente tener instalados al menos dos navegadores diferentes en tu ordenador? Indica el nombre del motor HTML y JavaScript de los navegadores que has instalado.

Es conveniente porque nos facilita la tarea de dividir nuestras tareas entre navegadores. Por ejemplo, se podría usar Firefox para el trabajo y Chrome para uso personal. No obstante, hoy en día los navegadores modernos incluyen opción de perfil que permiten dividir nuestro uso dentro del mismo navegador. A nivel de rendimiento y herramientas, casi todos ofrecen las mismas prestaciones.

Los motores HTML y JavaScript de los navegadores instalados son los siguientes:

  • Chrome: Actualmente Blink, pero empezó usando Webkit.
  • Mozilla Firefox: Gecko (desarrollado por Mozilla)
  • Safari: WebKit (desarrollado por Apple).
